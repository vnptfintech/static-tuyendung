Giới Thiệu

  • VNPT FinTech là đơn vị TIÊN PHONG được Tập đoàn Bưu Chính Viễn Thông Việt Nam (VNPT) giao nhiệm vụ triển khai và vận hành các dịch vụ tài chính số và được Ngân hàng Nhà nước cấp giấy phép cung ứng dịch vụ trung gian thanh toán. Tham gia vào VNPT FinTech đồng nghĩa với việc bạn sẽ tham gia vào 1 môi trường SỐ HOÁ và trở thành 1 thành viên của đại gia đình VNPT.

Mô Tả Công Việc

  • Với vai trò Thực tập sinh Quản trị hệ thống, bạn sẽ tham gia hỗ trợ vận hành và tối ưu hạ tầng hệ thống CNTT.
  • Công việc chính bao gồm:
  • Vận hành & giám sát hệ thống
  • Theo dõi tình trạng hệ thống:
  • Server (Linux/Windows)
  • Application
  • Database
  • Giám sát tài nguyên:
  • CPU, RAM, Disk, Network
  • Hỗ trợ xử lý sự cố hệ thống (downtime, performance)
  • Quản trị máy chủ & dịch vụ
  • Cài đặt, cấu hình:
  • Linux (Ubuntu/CentOS)
  • Web server (Nginx, Apache)
  • Hỗ trợ quản trị:
  • Application server
  • Database (MySQL, Oracle – cơ bản)
  • Triển khai & hỗ trợ hệ thống
  • Hỗ trợ deploy ứng dụng (Java, API services…)
  • Làm việc với:
  • File config
  • Environment setup
  • Hỗ trợ CI/CD cơ bản (nếu có)
  • Bảo mật hệ thống cơ bản
  • Cấu hình:
  • Firewall
  • SSL/TLS
  • Kiểm tra log, phát hiện bất thường
  • Hỗ trợ đảm bảo an toàn hệ thống
  • Tài liệu & phối hợp
  • Viết tài liệu:
  • Hướng dẫn vận hành
  • Runbook xử lý sự cố
  • Phối hợp với:
  • Dev
  • QA
  • ANTT

Yêu Cầu Ứng Viên

  • Chuyên môn & Học vấn:
  • Sinh viên năm cuối hoặc mới tốt nghiệp:
  • Công nghệ thông tin
  • Hệ thống thông tin
  • Mạng máy tính
  • Kiến thức & kỹ năng:
  • Có kiến thức cơ bản về:
  • Hệ điều hành Linux/Unix
  • Network (TCP/IP, DNS, HTTP/HTTPS)
  • Hiểu:
  • Client – Server
  • Web architecture
  • Kỹ năng & công cụ
  • Biết sử dụng:
  • Linux command line
  • SSH
  • Có kinh nghiệm (lợi thế):
  • Nginx / Apache
  • Docker (cơ bản)
  • Git
  • Kỹ năng mềm
  • Tư duy logic tốt
  • Chủ động học hỏi
  • Làm việc nhóm trong môi hình Agile/Scrum
  • Cẩn thận, phù hợp hệ thống tài chính
  • Khác
  • Đọc hiểu tài liệu tiếng Anh
  • Có định hướng theo System / DevOps / SRE

Quyền Lợi

  • Thu nhập: Có lộ trình trở thành nhân viên chính thức với thu nhập hấp dẫn.
  • Phát triển: Được đào về System thực tế, Kiến trúc Microservices
  • Thời gian làm việc: Từ thứ 2 đến thứ 6 hàng tuần, từ 8 đến 17h.
  • Địa điểm làm việc: 57A Huỳnh Thúc Kháng, Láng Hạ, Đống Đa, Hà Nội.
